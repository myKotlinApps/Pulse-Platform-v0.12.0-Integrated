# پروفایل نمایشی OBD-II

فایل مرجع: `shared/demo-profile.json`

## رفتار
- در اولین اجرای وب و هر اجرای Android که پروفایل دمو انتخاب شده باشد، Mock ELM327 خودکار وصل می‌شود.
- گیج‌های Speed و RPM از SVGهای استخراج‌شده از EPS استفاده می‌کنند.
- مقادیر با چرخه رانندگی قطعی تغییر می‌کنند و شامل توقف، دور آرام و شتاب‌گیری هستند.
- خواندن DTC دو کد نمونه P0133 و P0420 برمی‌گرداند.
- Mode 04 در دمو فقط وضعیت شبیه‌ساز را پاک می‌کند و نباید سهمیه خرید را مصرف کند.
- در وب، یک سفر نمونه با دو توقف برای تست 1×/2×/5×/10× Seed می‌شود.

## خروج از دمو
با انتخاب هر خودرو واقعی، اتصال Mock قطع و آخرین Live Snapshot پاک می‌شود. سپس کاربر باید Bluetooth Classic یا Wi-Fi TCP را متصل کند.
