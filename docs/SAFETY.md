# ایمنی فرمان پاک‌کردن DTC

1. Mode 04 فقط برای خطاهای عمومی مرتبط با آلایندگی موتور استفاده می‌شود.
2. پیش از ارسال، اتصال باید برقرار، موتور خاموش و سوییچ باز باشد.
3. برنامه RPM را درست پیش از ارسال فرمان دوباره می‌خواند و در صورت RPM بالاتر از 50 فرمان را رد می‌کند.
4. کاربر باید عبارت «پاک شود» را وارد کند.
5. پس از پاسخ مثبت 44، برنامه DTCها را دوباره می‌خواند.
6. Freeze Frame و مانیتورهای Readiness ممکن است پاک یا بازنشانی شوند.
7. پاک‌کردن ABS، SRS/Airbag، BCM، Immobilizer، J1939 DM3/DM11 و UDS 0x14 بدون پروفایل معتبر غیرفعال است.
8. هیچ فرمان Coding، Programming، Actuator، EEPROM، Flash یا تغییر کیلومتر در برنامه وجود ندارد.
