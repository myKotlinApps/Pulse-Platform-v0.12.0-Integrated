# یادداشت نشان‌های تجاری — v0.12.0

نشان‌های ایران‌خودرو، سایپا، زامیاد، بهمن، KMC، MVM، فونیکس، فردا موتورز، لاماری، دیار خودرو، ماموت خودرو، پارس‌خودرو، بهمن دیزل، ایران‌خودرو دیزل، مدیران خودرو و کرمان موتور در رابط فقط برای شناسایی برند/خودرو استفاده می‌شوند. مالکیت علائم برای صاحبان قانونی آن‌ها محفوظ است.

دارایی‌های v120 به PNG شفاف 640×640 نرمال شده‌اند. برای ایران‌خودرو و سایپا از مراجع PNG عمومی نشان فعلی؛ برای بهمن و ماموت از تصاویر هویت عمومی شرکت؛ و برای سایر برندها از دارایی‌های تأییدشده پیشین پروژه که با هویت عمومی سازنده تطبیق داده شدند استفاده شده است.
