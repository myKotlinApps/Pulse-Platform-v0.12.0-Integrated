# یادداشت نشان‌های تجاری — v0.13.0

فایل‌های پک `drawable-night-hdpi.zip` توسط کاربر ارائه شده‌اند. لوگوها فقط برای شناسایی برند خودرو در رابط کاربری استفاده می‌شوند و مالکیت علائم تجاری متعلق به صاحبان برندهاست. SVGهای قبلی به‌عنوان fallback فنی حفظ شده‌اند. نگاشت‌های فعال فقط برای مواردی انجام شده‌اند که تطبیق بصری نشان با برند موجود در پروژه مطمئن بوده است.
